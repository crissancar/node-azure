## Getting Started

This is an instructions on setting up the project locally.

### Prerequisites
Have **node** and **nvm** installed. Use version **20**.
```bash
$ nvm use 20.11.1
```

### Installation

1. Clone repository
```bash
$ git clone https://github.com/crissancar/node-azure.git
```
2. Install dependencies
```bash
$ npm install
```

### Running the app
```bash
$ npm run start:dev
```
